# تركيب البلاجن على WordPress — خطوتان فقط

## الخطوة 1 — نزّل الـ ZIP

اضغط على الرابط:
👉 https://github.com/growthack88/startup-book-landing/archive/refs/heads/main.zip

بعد التنزيل:
- افتح الـ ZIP
- ادخل مجلد `wp-plugin`
- هتلاقي مجلد اسمه `startup-book-landing`
- اعمل له ZIP منفرد (كليك يمين → Compress)

## الخطوة 2 — ارفعه على WordPress

```
WP Admin → Plugins → Add New → Upload Plugin
```

- اختار الـ ZIP اللي عملته
- اضغط Install Now
- بعدين Activate Plugin

## الخطوة 3 — إنشاء الصفحة

```
WP Admin → Pages → Add New
```

- اكتب العنوان: طريقك لبناء STARTUP
- من قائمة Template (يمين الشاشة) اختار: **Startup Book — Landing Page**
- اضغط Publish
- افتح الصفحة وشوف ✅
